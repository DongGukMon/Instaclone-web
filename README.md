# Instaclone

## Instaclone-Web
노마드코더의 [풀스택]인스타그램 클론코딩 강의를 들으며 진행한 프로젝트입니다.

## 기술 스택
- create react app
- javascript
- apollo client
- styled component
- react hooks

## 구현 범위
- Log In
- Sign Up
- Feed
- Create / Delete Comment
- Like / Unlike Photo
- Profile
